笔记使用方法：所有文字说明和可运行代码段全部以注释方式存在，其中，可运行代码段（知识点对应例程）在连续多个#下方，
                        如【# #####】或者【######】下方，运行时，多行选中需要运行的代码段，按【Ctrl】+【/】，即可将多行
                        注释变为可执行代码段，随后点击运行（绿色播放键）或者调试（绿色小虫），在运行栏或者调试栏观察实验现
                        象即可，建议自己跟着多打一下代码。运行完成后续要看下一段例程时，仍然多行选中执行过的运行的代码段，
                        按【Ctrl】+【/】将其重新变为注释后，不会影响其他程序段执行。
